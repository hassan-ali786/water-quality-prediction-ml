# Water Quality Prediction

An end-to-end machine learning web application that predicts whether water is safe for drinking using physicochemical water quality parameters and ensemble learning techniques.

---

## Overview

This project combines machine learning, data preprocessing, feature engineering, and an interactive web interface to analyze water quality and predict potability in real time.

The system allows users to enter water parameters through a clean Streamlit interface and instantly receive a prediction generated by a trained ensemble model.

## Dataset Features

| Feature | Description | WHO Safe Range |
|---|---|---|
| pH | Acidity or alkalinity level | 6.5 – 8.5 |
| Hardness | Mineral concentration in water | < 300 mg/L |
| Solids | Dissolved solid particles | < 500 ppm |
| Chloramines | Water disinfectant level | < 4 ppm |
| Sulfate | Sulfate concentration | < 250 mg/L |
| Conductivity | Electrical conductivity | < 400 μS/cm |
| Organic Carbon | Organic contamination level | < 2 ppm |
| Trihalomethanes | Chemical byproducts | < 80 μg/L |
| Turbidity | Water clarity level | < 1 NTU |

## Machine Learning Pipeline

### Data Preprocessing

- Missing value handling using mean imputation
- Data cleaning and validation
- Feature scaling using StandardScaler

### Feature Engineering

Created additional interaction-based features to improve model learning and prediction performance.

### Handling Imbalanced Data

- SMOTE applied to balance potable and non-potable classes

### Model Training

- Hyperparameter tuning using GridSearchCV
- Cross-validation for performance evaluation
- Ensemble learning for final prediction optimization

### Final Workflow

↓
